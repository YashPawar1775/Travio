# Travio: EJS, MongoDB, Maptiler
This is a full-stack web application built using Node.js and Express.js for backend API development and EJS with HTML, CSS, and JavaScript for the frontend. The application uses MongoDB as the database for efficient data management. It integrates the MapTiler API to provide interactive maps and location-based features. The project follows the MVC architecture and focuses on scalability, security, and a dynamic user experience.
